## Presentación

Este repositorio recoge un conjunto de recursos y herramientas digitales orientadas al ámbito de la **Formación Profesional**.

El objetivo principal es disponer de un espacio web sencillo, accesible y organizado, desde el que se puedan utilizar diferentes aplicaciones, formularios y utilidades para facilitar las labores administrativas del profesorado.
